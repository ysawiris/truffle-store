# truffle-store

This is a decentralized store app that sells shoes

## Installation

This application utlizes MetaMask. 
The contracts are deployed both on the Rinkeby Network and Ropsten Network


## Local

To run the application on a local network:

- Install Ganache 
- Install MetaMask 
```bash
npm run dev 
```

## Test Contract 
```bash
$ truffle test
```
